# foodopia-restaurant.github.io

I developed a dynamic restaurant website using Frontend technologies such as HTML, CSS, and Javascript, along with Backend technologies like PHP. The website leverages a MySQL database to store and manage user data, menus, orders, and reservations. Users can easily navigate the website to browse the menu, place orders, and make table reservations. Administrators have access to functionalities for managing orders, updating payment statuses, and handling customer inquiries. The website offers a seamless and user-friendly experience for both customers and staff.
